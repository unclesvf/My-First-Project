# Math Training and Assessment App

A Python-based interactive math learning application that progressively adapts to the user's skill level.

## Features
- Progressive difficulty levels:
  1. Simple addition and subtraction
  2. Multi-digit arithmetic (up to 3 digits)
  3. Multiplication (numbers up to 12)
  4. Division (denominators up to 12, numerators up to 144)
- Adaptive learning system
- Performance tracking
- Immediate feedback

## How to Run
1. Ensure Python 3.x is installed
2. Run `python math_trainer.py`
