# Export Prompts

## Description
Quick export of image generation prompts in various formats suitable for external services like Higgsfield.ai, Nano Banana Pro, or other AI image generators.

## When to Use
- To use prompts with Higgsfield web UI (unlimited with Ultimate plan)
- To prepare prompts for batch upload to external services
- To create spreadsheets for tracking/organizing prompts
- To share prompts with collaborators
- To backup prompt library

## Capabilities
- Exports all 265 image prompts
- Multiple output formats (TXT, CSV, clean)
- 60-dash separators for easy parsing
- Filter by lesson number or era
- Includes metadata (lesson title, chapter, era)
- Auto-opens output folder

## Process

### Step 1: Determine Export Parameters
Ask user for:
1. **Format**: Full (with metadata), Clean (prompts only), or CSV
2. **Filter**: All prompts, specific lesson, or specific era
3. **Output name**: Custom or auto-generated with timestamp

### Step 2: Run Export Script
```bash
cd "C:\Users\scott\My-First-Project\My-First-Project\history-lesson-creator\scripts"

# Full export with metadata
python export_prompts_for_higgsfield.py

# Clean prompts only
python export_prompts_for_higgsfield.py --format prompts

# CSV format
python export_prompts_for_higgsfield.py --format csv

# Filter by lesson
python export_prompts_for_higgsfield.py --lesson 10

# Filter by era
python export_prompts_for_higgsfield.py --era "1776"
```

