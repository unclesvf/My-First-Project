# Lesson Exports for AI Assistants

## What's in this folder

**LESSON-SUMMARY.md** - Overview of all 50 lessons with titles and descriptions

## How to use

1. Upload LESSON-SUMMARY.md to Grok, Gemini, or ChatGPT
2. Ask the AI to visit https://auntsvaluefarm.com/history/ to view the full lessons
3. Request image prompts for specific lessons you want to illustrate

## Sample prompt to use

"I have 50 American History lessons for 7th graders. Here's the summary of all lessons. Please visit https://auntsvaluefarm.com/history/lesson/lesson-1/ to view Lesson 1, then create detailed image generation prompts for each chapter that would help illustrate the story."
